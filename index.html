ESTABLECER LA DIRECCIÓN DEL EJE PRINCIPAL DEL CONTENEDOR FLEXIBLE

La propiedad flex-direction permite establecer la dirección del eje principal del contenedor flexible. Los elementos flexibles se colocan a en la dirección del eje principal.

EJE PRINCIPAL DE IZQUIERDA A DERECHA

HTML

<div class="contenedor">
    <div>E1</div>
    <div>E2</div>
    <div>E3</div>
</div>

CSS

.contenedor{
    display: flex;
    flex-direction: row; [VALOR  POR DEFECTO] 
}

EJE PRINCIPAL DE DERECHA A IZQUIERDA

HTML

<div class="contenedor">
